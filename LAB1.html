Programação na Web

Na Escola do Comercio de Lisboa, a na disciplina de Tecnologia 1 no 1ºATIS, iremos aprender Programação Web 😅 e aboradremos os principios básicos associados ao desenvolvimento para a aWeb, apenas do lado front-end,lado visual. Também sabemos que existe um lado back-end,que é o lado do servidor, mas nao iremos abordar esse lado.

Aprender-se-ão as seguintes tecnologias 🌐:

  • HTML
  • CSS
  • Java script

De forma a consolidar todos os conhecimentos, serão feitos alguns laboratorios.
